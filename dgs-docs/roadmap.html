Roadmap

These roadmap notes only capture improvements strongly suggested by the current codebase and tooling shape.

High-value near-term improvements

  • add more automated tests for graph traversal, JSON import/export, and transcript rebuilding
  • improve validation and diagnostics around malformed graphs
  • harden package separation for the AI extension
  • continue improving editor polish and graph safety around Undo/Redo
  • improve onboarding visuals with screenshots and guided examples

Medium-term improvements

  • formalize custom node extension patterns
  • reduce hard-coded resource path assumptions
  • expand transcript and reporting features
  • add more explicit upgrade support if graph DTOs or node schema evolve
